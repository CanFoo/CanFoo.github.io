面向对象编程

发表于 2014-09-25

几个面向对象的概念

对象：一个泛指，万物皆对象。
类：把对象具体的细分。
实例：某一个类别中具体的事物。
例子：自然界中万物都是对象，想要认知自然界，我们需要把自然界万物按照特征分为不同的类别：人类、动物类、植物类… 每一个人都是人类中的一个实例。
面向对象（JavaScript）:把JS分成几个类，然后在每一个类中分别拿出一个具体的实例去学习研究，当这个实例研究清楚后，我们就可以认为当前类的其他实例也拥有这些知识（特点）

JS面向对象几种模式

  • 单例模式（只能返回一个实例）:1.可以起到分组的作用；2.防止冲突

THIS关键字总结

发表于 2014-09-22

this关键字

this表示当前运行方法的主体，函数中的this指向和当前函数在哪定义的话或者在哪执行都没有任何的关系。

this的上下文

  • 自制行函数里面的this永远都是window
    var inner = "window";  
var obj = {inner : "obj",  
          fn : (function () {console.log(this.inner)})()  
}  
/*上面浏览器在运行该程序时，会自动运行obj.fn里面的方法，因为obj.fn是一个自制行函数，当执
行该函数时，程序会输出window。
为什么输出不是obj？
因为人家规定自制行函数里面的this是window,所以其实this.inner就是window.inner，因此这个
inner是定义在全局变量的，它的值是"window"。*/

作用域与闭包

发表于 2014-09-20

作用域

  • 全局作用域与私有作用域

    1. JS运行在浏览器中的客户端脚本编程语言，当浏览器加载HTML页面的时候首先会提供一个供JS代码的环境，我们把它称之为全局作用域。
    2. 函数执行的时候，目的是为了让定义时候存储的那些代码字符串变成真正的代码去执行，因此执行的时候会产生一个私有作用域，供这些代码去执行。

  • 作用域链：在私有作用域中，JS代码从上到下执行，如果遇到了一个变量，首先看是否是自己私有的变量，如果是私有的，那么我们接下来的所有操作都是用的自己私有的，和外面的没有任何的关系；如果不是自己私有的，则去当前作用域的上一级作用域中查找，如果上一级有，那么接下来操作的，都是操作上级作用域的变量；如果上级也没有，则继续向上查找，一直找到window为止。

预解释

发表于 2014-09-19

预解释

预解释（变量提升）：在当前的作用域中，JS代码从上到下执行的时候，浏览器会默认的先把所有带var/functon关键字进行提前的声明或者定义。

  1. 声明：告诉浏览器在当前的作用域中有一个名字叫做num(var num;)，只声明没有定义的话，默认的值是undefined
  2. 定义：给变量赋值（eg num=10;）
  • 在预解释阶段，带var关键字的只是提前的声明，只有在JS从上到下执行的时候才会定义赋值
